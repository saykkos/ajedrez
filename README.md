# Ajedrez Multiplayer

Servidor de señalización WebRTC para el juego de ajedrez.

## Configuración

1. Abre un terminal en `c:\Users\xxoga\Desktop\Ajedrez`
2. Ejecuta:

```bash
npm install
```

## Ejecutar

```bash
npm start
```

Luego abre en el navegador:

```text
http://localhost:8080/chess.html
```

## Uso

- `Crear partida`: genera un código de sala.
- `Unirse a partida`: usa el código de sala para conectar con otro jugador.
- El servidor transmite la señalización WebRTC entre los dos navegadores.
